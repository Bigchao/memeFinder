3. **关联性分析**
   - 交易关系网络
   - 资金流向追踪
   - 共同行为模式
   - 时序相关性

### 2.2 评分系统
- **基础指标**
  ```
  基础分 = 收益率*0.4 + 成功率*0.3 + 交易规模*0.2 + 活跃度*0.1
  ```

- **加权因子**
  ```
  最终分 = 基础分 * (1 + 风险调整系数) * (1 + 创新因子)
  ```
